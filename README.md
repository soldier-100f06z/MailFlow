## ✨ Features

- **3 tabs** — Templates, Server, Proxy
- **10+ ready phishlets** with preview images and hover-to-select interaction
- **Real-time stats** — tracks sent / total count in the header
- **SMTP server configuration** — Host, Port, Login, Password with connection check
- **Mass Upload** button for bulk data import
- **Input field** supporting up to 10,000 lines of data
- **Fully dark UI** with custom color palette, gradients, and accent highlights

## 🚀 Installation

### Download

1. Go to the [**Releases**](../../releases) page
2. Download the latest `MailFlowPro-x.x.x.msi` file
3. Run the installer and follow the on-screen steps
4. Launch **MailFlow Pro** from the Start Menu or Desktop shortcut

> No Python or any additional software required — the installer includes everything.
